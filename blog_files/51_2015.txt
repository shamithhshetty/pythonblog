<h3 class="post-title entry-title">
<a href="https://pythoninsider.blogspot.com/2015/06/python-350b2-released.html">Python 3.5.0b2 released</a>
</h3>
<div class="post-header">
<div class="post-header-line-1"></div>
</div>
<div class="post-body entry-content">
Python 3.5.0b2 is now <a href="https://www.python.org/downloads/release/python-350b2/">available for download.</a><br/>
<br/>
Python 3.5.0b1 had a major regression
            (see <a class="moz-txt-link-freetext" href="http://bugs.python.org/issue24285">http://bugs.python.org/issue24285</a> for more information)
            and as such was not suitable for testing Python 3.5. 
            Therefore we've made this extra beta release, only a week
            later.  Anyone trying Python 3.5.0b1 should switch
            immediately to testing with Python 3.5.0b2.<br/>
<br/>
<br/>
Python 3.5 has now entered "feature freeze".  By default new
            features may no longer be added to Python 3.5.  (However,
            there are a handful of features that weren't quite ready for
            Python 3.5.0 beta 2; these were granted exceptions to the
            freeze, and are scheduled to be added before beta 3.)<br/>
<br/>
This is a preview release, and its use is not recommended
            for production settings.
